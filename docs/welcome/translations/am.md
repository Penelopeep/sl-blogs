# **рІерѕ░ріљрІх рѕ░ріљрІх**

рІѕрІ░ рідрЇірѕ┤рѕІрІі рѕ░ріљрІх рѕЏрІЋріерѕЇ ріЦріЋрі│ріЋ рЅарІ░рѕЁріЊ рѕўрїА **рЇѕрѕЇрїЇ**.\
рІеріЦріЏ рѕўрѕ│рѕфрІФрІјрЅй ріЦріЋрІ▓рѕєріЉ рІерЅ░рЅђрІерѕ▒ ріЊрЅИрІЇ _рЅарїБрѕЮ рѕірЅарїЁ рІерѕџрЅйрѕЇ_, *рѕърІ▒рѕЇ*ріЦріЊ _рЅ░рѕхрѕЏрѕџ_ рІѕрІ░ рѕЦрѕФрІј рЇЇрѕ░рЅх.

## ріарїарЅЃрѕІрІГ ріЦрІГрЅ│

- **рѕЏрѕѕрЅѓрІФ рІерѕїрѕѕрІЇ рЅЁрїБрЅхріЋ**рІерѕџрІФрІФрІЎрЅх рїѕрїЙрЅй рѕўрѕЇрІЋріГрЅх.
- **рѕърІ▒рѕЇ ріЋрІхрЇЇ**: рЅ░рѕ░ріфрІјрЅйріЋ ріЦріЊ рѕърїЂрѕјрЅйріЋ рЅарѕўрїарЅђрѕЮ рЅ░рїЇрЅБрѕЕріЋ рІФрѕ░рЇЅ.
- **рѕЏрѕЁрЅарѕерѕ░рЅЦ-рЅ░рїјрІхрЅирѕЇ**: ріЦріЋрІ░ ріЦрѕГрѕхрІј ріФрѕЅ рІеріЃрІГрѕЇ рЅ░рїарЅЃрѕџрІјрЅй рїЇрЅЦрѕерѕўрѕЇрѕх рїІрѕГ рЅ░рїѕріЋрЅЦрЅирѕЇ.
