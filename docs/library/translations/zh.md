# **Seelen UI库（SLU-LIB）**

## 概述

Seelen UI库是用于为Seelen UI创建小部件和扩展的官方开发工具包。 它提供：

- 类型定义以一致开发
- 无缝集成的API抽象
- 小部件互操作性的标准化接口

## 核心组件

1. **类型模块**\
   基础类型定义和接口

2. **API模块**\
   小部件开发的实施和实用程序

## 入门

```bash
npm install @seelen-ui/lib
```
