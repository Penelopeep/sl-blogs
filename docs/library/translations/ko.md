# **Seelen UI 라이브러리 (SLU-LIB)**

## 개요

Seelen UI Library는 Seelen UI의 위젯 및 확장을 만들기위한 공식 개발 툴킷입니다.
그것은 제공합니다 :

- 일관된 개발을위한 정의 유형
- 원활한 통합을위한 API 추상화
- 위젯 상호 운용성을위한 표준화 된 인터페이스

## 핵심 구성 요소

1. **유형 모듈**\
   기본 유형 정의 및 인터페이스

2. **API 모듈**\
   위젯 개발을위한 구현 및 유틸리티

## 시작하기

```bash
npm install @seelen-ui/lib
```

**메모**: 자세한 문서는 전용 섹션에서 제공됩니다. [유형](./library-types) 그리고
[API](./library-api).
