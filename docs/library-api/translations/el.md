# **Βιβλιοθήκη SLU: Αναφορά API**

Η μονάδα API παρέχει συγκεκριμένες υλοποιήσεις και κατηγορίες κοινής ωφέλειας
Περίληψη Το API Applice API, που χρησιμεύει ως φιλική προς τον προγραμματιστή
πρόσοψη για ενσωμάτωση γραφικού στοιχείου.

## **Σκοπός**

- Εφαρμόζει διεπαφές από `/types` με έτοιμες τάξεις
- Περιλήψεις χαμηλού επιπέδου API πολυπλοκότητες
- Παρέχει μεθόδους χρησιμότητας για κοινές λειτουργίες
- Χρησιμεύει ως το κύριο στρώμα αλληλεπίδρασης μεταξύ των widgets και του SLU
  εφαρμογή

## **Χρήση**

Μπορείτε να χρησιμοποιήσετε τη βιβλιοθήκη ανάλογα με τις ανάγκες. Ακολουθεί ένα
βασικό παράδειγμα:

```ts
import { StructClass } from "@seelen-ui/lib/api";

const widgetApi: StructClass = await StructClass.geAsync();
const something = widgetApi.getSomething();
const somethingAsync = await widgetApi.getSomethingAsync();

await widgetApi.doSomething();
```

## **Σημαντικές σημειώσεις:**

1. Αυτή η βιβλιοθήκη API λειτουργεί μόνο όταν εκτελείται μέσα σε μια εκτέλεση
   Widget Seelen UI περιβάλλο
2. Η προσπάθεια να το χρησιμοποιήσετε σε node.js ή τυποποιημένα πλαίσια
   προγράμματος περιήγησης θα ρίξει λάθη
3. Τα widgets που χρησιμοποιούν αυτό το API πρέπει να το ενημερώσουν για να
   λειτουργήσουν σωστά με τελευταία έκδοση του Seelen UI
